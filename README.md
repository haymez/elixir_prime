# Prime

Elixir app that finds prime numbers concurrently. 

## Installation

```
git clone https://github.com/haymez/elixir_prime.git
cd elixir_prime
mix deps.get
```

## Usage

```elixir
iex> Prime.find_primes(1..100)
iex> [2, 3, 5, 7 ...]
```
